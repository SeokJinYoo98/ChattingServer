# YuJanggiServer

C# `TcpListener`와 `TcpClient`를 사용해서 구현하는 간단한 콘솔 기반 채팅 서버 프로젝트입니다.

## 목표

- 여러 클라이언트의 TCP 접속을 수락합니다.
- 클라이언트가 보낸 메시지를 서버에서 처리합니다.
- 접속한 클라이언트 간 메시지를 브로드캐스트합니다.
- 기본적인 접속 종료와 예외 처리를 다룹니다.

## 개발 환경

- .NET SDK
- C#

## 실행 계획

1. 서버 콘솔 앱 생성
2. `TcpListener` 기반 접속 수락 구현
3. 클라이언트별 비동기 메시지 수신 구현
4. `TcpClient` 목록 관리 및 메시지 브로드캐스트 구현
5. 간단한 콘솔 클라이언트 추가

## 참고

이 프로젝트는 TCP 소켓 프로그래밍 학습용 예제입니다. 인증, 암호화, 영속 저장소 같은 운영용 기능은 포함하지 않습니다.

## PR 전략

- 작업 단위는 작게 유지합니다. 예를 들어 서버 초기화, 클라이언트 접속 처리, 메시지 브로드캐스트, 콘솔 클라이언트 추가를 각각 분리합니다.
- PR 제목은 변경 목적이 드러나도록 작성합니다.
- PR 본문에는 변경 내용, 테스트 방법, 남은 작업을 간단히 적습니다.
- 동작 변경이 있는 PR은 실행 방법과 확인 결과를 함께 남깁니다.
- 리팩터링과 기능 추가는 가능하면 같은 PR에 섞지 않습니다.
- `main` 브랜치는 실행 가능한 상태를 유지하고, 기능 작업은 별도 브랜치에서 진행합니다.
